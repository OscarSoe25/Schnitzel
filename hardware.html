Hardware Page

My hardware page will be describing what the hardwares' inside our computers do and a little bit of other useful information.


Motherboard

This is a motherboard. A motherboard is the main printed circuit board with the components of a computer or other device, and other circuit boards can be connected. CPUs, RAMS and other hardware can connect to a motherboard and the motherboard communicates to each of the components

CPU

A cpu stands for central processing unit is an electronic circuitry that carries out instructions to the computer by using basic arithmetic, logical, control and input/output with specific instructions.

RAM

When we talk about RAM, we aren't talking about the car company. We are talking about Random Access Memory. RAM is a place where the computer’s memory can be accessed. The more RAM you have in a computer the more bytes a program can access.

Hardrive

A hard drive is a storage device inside a computer or laptop. Everything on the computer is stored in the hard drive, it is the hard drives’ responsibility to store things.
